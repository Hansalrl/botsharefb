# bot_share_facebook
a bot to share Facebook posts.
# install & clone
```BASH
cd
pkg install git -y
pkg install python -y
git clone https://github.com/Latip176/bot_share_facebook
cd bot_share_facebook
python3 -m pip install -r requirements.txt
```
# running / menjalankan
```BASH
cd $HOME/bot_share_facebook
python share.py
```
